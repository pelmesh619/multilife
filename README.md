# multilife

multilife - многопользовательская игра, вдохновленная клеточным автоматом "Жизнь" Джона Конвея

В основе игры лежит классический клеточный автомат, мир которого представляет собой двумерную сетку клеток. Каждая клетка может находиться в одном из нескольких состояний. Сервер циклически обновляет состояние мира по заданным правилам, а подключенные клиенты могут в реальном времени визуализировать процесс и влиять на него

Проект основан клиент-серверной архитектуре
